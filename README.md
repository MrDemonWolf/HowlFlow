# HowlFlow

ADHD daily companion mobile app with wolf theming. Local-first architecture — your data stays on your device.

## Apps

- **`apps/mobile`** — iOS app built with Expo SDK 55 + NativeWind v5
- **`apps/docs`** — Marketing site, Privacy Policy, and Terms of Service

## Getting Started

```bash
pnpm install
pnpm dev:mobile    # Start Expo dev server
pnpm dev:docs      # Start docs dev server
```
