# Welcome to HowlFlow

HowlFlow is an ADHD productivity app for iOS with a wolf theme. It provides structured daily scheduling, focus timers, brain dumps, and a gamified XP system — all designed for teens with ADHD.

## What's in the Docs

- [Getting Started](/docs/getting-started) — Set up the development environment and run the app
- [Features](/docs/features) — Explore Schedule Blocks, Focus Timer, Brain Dump, and more
- [Architecture](/docs/architecture) — Understand the monorepo layout, state management, and theming

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Framework | Expo SDK 55 / React Native 0.83 |
| Styling | NativeWind v5 (Tailwind for RN) |
| State | Zustand + MMKV |
| Navigation | Expo Router (file-based) |
| Notifications | expo-notifications |
| Native Modules | expo-modules-core (Swift) |
| Docs | Fumadocs + Next.js |
| Monorepo | Turborepo + pnpm |
